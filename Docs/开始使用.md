## 🚀 立即开始

### 第一步：打开 Xcode

1. 打开 Xcode (需要 15.0 或更高版本)
2. 选择 `File > New > Project`
3. 选择 `iOS` > `App`

### 第二步：配置项目

- **Product Name**: HabitKit
- **Interface**: SwiftUI
- **Storage**: SwiftData
- **Language**: Swift
- **Minimum Deployments**: iOS 17.0

### 第三步：导入文件

1. 删除 Xcode 自动生成的 `ContentView.swift`
2. 将当前目录下的所有 `.swift` 文件拖入 Xcode
3. 确保勾选 `Copy items if needed`
4. 保持文件夹结构

### 第四步：运行

1. 选择模拟器（推荐 iPhone 15 Pro）
2. 点击运行按钮 (▶️) 或按 `Cmd + R`
3. 等待编译完成

## ❓ 常见问题

**Q: 为什么需要 iOS 17+？**  
A: 项目使用了 SwiftData，这是 iOS 17 的新特性。如果需要支持更低版本，可以改用 Core Data。

**Q: 可以在真机上运行吗？**  
A: 可以！只需要在 Xcode 中配置开发团队即可。

**Q: 如何备份数据？**
